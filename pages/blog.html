Important FAQ

What is the Document Object Model(DOM)?

DOM is a programming interface for web documents.It represents the structure of a web page as a tree of objects and nodes.

How do you select an element from the DOM?

We can select an element from the DOM by using browser's DOM API.Some common methods are getElementById() ,getElementsByClassName(), querySelector() and querySelectorAll()

What is event delegation in the context of the DOM,and why is it usefull?

Event delegation is an efficient way to manage event handling in the dom by attaching a single event listener to a parent element instead of every child element of parent.It is useful because event delegation simplifies handling of dynamic added elements,reduces code redundancy.

How do you manipulate an element's attributes and styles using the DOM?

There are various ways to manipulate an element's attributes and styles using the DOM.Some common attribute methods are setAttribute() ,removeAttribute() etc.We can use style.propertyName,classList.add() ,classList.remove() etc to style dom element.
